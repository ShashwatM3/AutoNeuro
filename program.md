# Program

## Overview

This document outlines the AutoResearch-style multi-agent ML pipeline. Details will be filled in as the program evolves.

## Agent Roles (Coding, Research, Contact)

The **Coding** agent implements changes in the codebase. The **Research** agent proposes directions from literature and notes. The **Contact** agent interfaces with humans when decisions or approvals are required.

## Metrics and Evaluation

Primary and secondary metrics will be recorded per experiment. Evaluation runs will populate structured result tables for comparison across runs.

## Safety / Human-in-the-loop

Critical or ambiguous steps will require human review before execution. Safety constraints and escalation paths will be documented alongside agent behavior.
